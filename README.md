# Web Scraping Project

A minimal Python web scraper that extracts structured data from websites.
before you steal the project at least drop a star <3

## Setup

1. Install dependencies:

```bash
pip install -r requirements.txt
```

## Run

```bash
python scraper.py
```

## Output

The scraper will generate:

- `scraped_data.csv` - Data in CSV format
- `scraped_data.json` - Data in JSON format

## Customize

Edit `scraper.py` to:

- Change the target URL
- Modify data extraction logic
- Add more fields to extract
